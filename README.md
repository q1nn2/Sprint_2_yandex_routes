# Яндекс Маршруты — Sprint 2

## Описание

Ручное тестирование веб‑сервиса **Яндекс Маршруты**: тест-анализ, классы эквивалентности и граничные значения, тест-кейсы и баг-репорты.

Цель работы — оценить качество реализации валидации полей времени и адресов, а также корректность расчёта стоимости и времени поездки на собственном автомобиле. На основе результатов сформированы выводы о готовности продукта к релизу.
