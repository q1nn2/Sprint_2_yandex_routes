# КЭ и ГЗ для расстояний и интервалов времени

Вкладка Google Sheets: «Этап 2. КЭ и ГЗ».

## Расстояние между адресами

| Группа проверок            | Название класса | Примеры                                 |
|----------------------------|-----------------|-----------------------------------------|
| Расстояние = 0 (A = B)    | Расстояние = 0  | Усачева, 3 → Усачева, 3                |
| Расстояние > 0 (A ≠ B)    | Расстояние > 0  | Усачева, 3 → М. Пироговская, 25        |

## Интервалы времени начала движения

| Интервал                   | Условие         | Примеры значений времени начала |
|---------------------------|-----------------|----------------------------------|
| Ночь                      | 00:01–08:00     | 00:01, 07:59, 08:00             |
| Утро                      | 08:01–12:00     | 08:01, 10:10, 11:59, 12:00      |
| День                      | 12:01–18:00     | 12:01, 15:23, 17:59, 18:00      |
| Вечер                     | 18:01–22:00     | 18:01, 20:12, 21:59, 22:00      |
| Поздний вечер             | 22:01–00:00     | 22:01, 23:15, 23:59, 00:00      |
