# Тест-кейсы: расчёт стоимости и времени поездки на своём автомобиле

Группа из вкладки Google Sheets «Этап 3. Тест-кейсы»: **Расчёт стоимости поездки на собственном автомобиле** (`T60–T65`).

Общие предусловия для всех кейсов:

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

---

## T60 — Расчёт стоимости поездки на собственном автомобиле при расстоянии равном 0

- **ID**: T60  
- **Группа кейсов**: Расчёт стоимости поездки на собственном автомобиле  

- **Шаги**:
  1. Заполнить поле Часы значением `08`.  
  2. Заполнить поле Минуты значением `10`.  
  3. Заполнить поле Откуда значением `Усачева, 3`.  
  4. Заполнить поле Куда значением `Усачева, 3`.  
  5. Кликнуть на кнопку «Свой».  
  6. Кликнуть на иконку «Автомобиль».

- **Ожидаемый результат**:  
  Получен расчёт стоимости и времени поездки.  
  - Стоимость = 0 рублей.  
  - Время в пути = 0 мин.

- **Окружение**:  
  - Браузер: Google Chrome v.144.0.7559.133  
  - ОС: Windows 11 Pro 21H2

- **Статус**: `Passed`  
- **Связанные баг-репорты**: нет
