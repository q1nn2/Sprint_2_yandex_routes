## T49 — Валидация поля Откуда: проверка принятия системой адреса из «Списка доступных адресов»

- **ID**: T49  
- **Группа кейсов**: Откуда  
- **Предусловия**:  
  - Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

- **Шаги**:
  1. Заполнить поле Откуда значением `Усачева, 3`.  
  2. Снять фокус с поля.

- **Ожидаемый результат**:  
  Значение принимается системой, ошибки нет.

- **Окружение**:  
  - Браузер: Google Chrome v.144.0.7559.133  
  - ОС: Windows 11 Pro 21H2

- **Статус**: `Passed`  
- **Связанные баг-репорты**: нет
