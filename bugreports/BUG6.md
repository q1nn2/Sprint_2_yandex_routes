# BUG6 — Появляется ошибка при вводе однозначных чисел в поле «Минуты»

- **ID**: BUG6  
- **Статус**: Открыт  
- **Приоритет**: Критический  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Минуты значением `02`.  
2. Снять фокус с поля.

## Ожидаемый результат

Система принимает однозначные числа, ошибки нет.

## Фактический результат

Система не принимает однозначные числа, появляется ошибка «Вы ввели некорректное время».

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
