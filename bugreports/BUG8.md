# BUG8 — Появляется некорректное сообщение об ошибке в поле «Откуда» при вводе адреса не из списка доступных адресов

- **ID**: BUG8  
- **Статус**: Открыт  
- **Приоритет**: Желательный  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Откуда значением `Ленина Матросова, 20`.  
2. Снять фокус с поля.

## Ожидаемый результат

Появляется ошибка «Вы ввели некорректный адрес».

## Фактический результат

Появляется ошибка «Вы ввели некорректное время».

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
