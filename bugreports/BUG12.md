# BUG12 — Не удаляется пробел перед адресом в поле «Куда»

- **ID**: BUG12  
- **Статус**: Открыт  
- **Приоритет**: Желательный  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Куда значением `" Хамовнический Вал, 34"`.  
2. Снять фокус с поля.

## Ожидаемый результат

Пробел перед адресом удаляется, значение принимается системой, ошибки нет.

## Фактический результат

Пробел перед адресом не удаляется, значение принимается системой.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
