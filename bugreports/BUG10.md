# BUG10 — Не удаляется пробел после адреса в поле «Откуда»

- **ID**: BUG10  
- **Статус**: Открыт  
- **Приоритет**: Желательный  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Откуда значением `"Усачева, 3 "`.  
2. Снять фокус с поля.

## Ожидаемый результат

Пробел после адреса удаляется, значение принимается системой, ошибки нет.

## Фактический результат

Пробел после адреса не удаляется, но значение принимается системой.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
