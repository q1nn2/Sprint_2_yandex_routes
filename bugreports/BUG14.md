# BUG14 — Не появляется сообщение об ошибке в поле «Куда» при пустом вводе

- **ID**: BUG14  
- **Статус**: Открыт  
- **Приоритет**: Критический  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Поставить фокус в поле Куда.  
2. Оставить поле Куда пустым.  
3. Снять фокус с поля.

## Ожидаемый результат

Поле подсвечивается красным, появляется ошибка «Вы ввели некорректный адрес».

## Фактический результат

Система принимает значение, ошибки нет.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
