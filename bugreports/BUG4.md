# BUG4 — Не появляется ошибка при оставлении пустым поля «Часы»

- **ID**: BUG4  
- **Статус**: Открыт  
- **Приоритет**: Критический  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Поставить фокус в поле Часы.  
2. Оставить поле Часы пустым.  
3. Снять фокус с поля.

## Ожидаемый результат

Поле подсвечивается красным, появляется ошибка «Вы ввели некорректное время».

## Фактический результат

Система принимает значение, ошибки нет.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2
